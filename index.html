Funciones de Uso Común

Hemos denominado este apartado como funciones de uso común, ya que al igual que en Matlab, las funciones que aquí se presentan no son extrictamente implementadas para el procesamiento de señales, sino en general como herramientas comunes cuando se trabaja con elementos matriciales y vectoriales, o donde se involucra el análisis con polinomios.

Método size

La función size( ) retorna el número de filas y columnas presentes en una matriz de Python.

Ejemplo: sea la siguiente matriz de 2x3.

Mat = [[1,2,3],[4,5,6]]

Obtenemos las filas y columnas de la matriz con size()

fil, col = size(Mat)

Podemos visualizar el resultado en pantalla con:

print 'Filas =',fil,' Columnas =',col

Función poly

El método poly al igual que en Matlab, retorna en un vector los coeficientes del polinomio cuyas raices son las mismas que se le pasan como argumento.

Ejemplo: Sea el vector con las raices raices = [-0.5+3*1j, -0.5-3*1j, 4] se obtienen los coeficientes del polinomio como:

coefs = poly( raices )

Podemos imprimir dichos coeficientes en pantalla como: print 'Coeficientes =, coefs

Obteniendo como resultado: Coeficientes = [1, -3, 5.25, -37] que equivale a s^3 -3s^2 +5.25s -37

Función polyval

La función polyval al igual que en Matlab, realiza la evaluación de un polinomio dados sus coeficientes, para un valor especificado como segundo parámetro.

Ejemplo: Sea el polinomio x^3 - 3x^2 + 5.25x - 37 y queremos evaluarlo en x = 2 el procedimiento es entonces, almacenar los coeficientes en un array y luego indicarle a la función en su segundo parámetro, el valor que queremos evaluar.

Coefs = [ 1,-3,5.25,-37 ]
Val = 2
Resul = polyval( Coefs, Val )

print 'Resultado =', Resul

Función linspace

La función linspace al igual que su homóloga en Matlab, retorna un vector con sus elementos linealmente espaciados, es decir, si queremos un vector de 500 valores cuyo rango esté entre 1 y 50, la sintaxis sería la siguiente:

Vec = linspace(1, 50, 500)

Ya que el prototipo de la función es: vector = linspace(x1, x2, num_elements)

Función ones

En este módulo, la función ones retorna una matriz de 1 (unos) de MxN elementos.

Ejemplo: si deseamos una matriz compuesta de 1s de 20 filas por 30 columnas (20x30 elementos), la sintaxis sería la siguiente:

MatrizUnos = ones(20,30)

Si utilizamos size() para obtener sus dimensiones, el resaultado sería:

print 'Dimensiones =',size(MatrizUnos)

Función zeros

La función zeros retorna una matriz de 0s (ceros) de MxN elementos.

Ejemplo: si deseamos una matriz compuesta de 0s de 30 filas por 50 columnas (30x50 elementos), la sintaxis sería la siguiente:

MatrizCeros = zeros(30,50)

Si utilizamos size() para obtener sus dimensiones, el resaultado sería:

print 'Dimensiones =',size(MatrizCeros)

Función matInit

Se puede considerar a la función matInit como un complemento de zeros y ones ya que retorna una matriz de MxN elementos cuyos valores iniciales se indican como el tercer parámetro de la función, soportando además valores complejos.

Ejemplo: si deseamos una matriz de 4 filas por 5 columnas (4x5 elementos), cuyos valores iniciales sean el número 98, la sintaxis es la siguiente:

valInicial = 98

Matriz = matInit(4,5,valInicial)

Si utilizamos printMatrix() se imprimirán los valores de la matriz en pantalla:

printMatrix( Matriz )

Función round_vec

Esta función aplica el método round() a todos los elementos de un array, que se le pasa como argumento.

Ejemplo: Sea la matriz Matriz = [[2.32, 4.35, 5.64],[3.21, 6.43, 7.89]] aplicando el método round_vec() el resultado sería:

MatrizR = round_vec(Matriz)

MatrizR = [ [2, 4, 6],[3, 6, 8] ]
